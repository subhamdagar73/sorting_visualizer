# Sorting_Visualiser

An interactive, educational sorting algorithm visualizer built with HTML, CSS, and JavaScript. Watch how different sorting algorithms work step by step, control the speed, and compare their time and space complexities in real time. It also supports customizable array sizes, dynamic color themes for enhanced accessibility, and intuitive controls for pausing, resuming, and resetting visualizations on demand.
